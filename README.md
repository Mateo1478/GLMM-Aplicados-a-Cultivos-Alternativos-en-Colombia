# GLMM-Aplicados-a-Cultivos-Alternativos-en-Colombia
Repositorio oficial de la tesis de Maestría en Ciencias – Estadística (Universidad Nacional de Colombia). Contiene la documentación, bases de datos, scripts en R y resultados asociados a la aplicación de Modelos Lineales Generalizados Mixtos (GLMM) para el análisis de cultivos alternativos a economías ilícitas en Colombia.
